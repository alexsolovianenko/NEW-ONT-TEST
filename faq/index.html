Frequently Asked Questions

Ontario Tests converts all testable material into comprehensive practice exams. Our platform systematically covers every concept teachers expect you to know, eliminating the guesswork from quiz/test/exam prep.

Students upload test material, which our OCR API converts to text. Our custom LLM—built with Pandas, NumPy, and Hugging Face Transformers—analyzes the content to generate every possible question variation for the unit. Fun fact: each test is only 5KB, with all Ontario Tests combined under 3MB.

Our LLM operates at ~90% accuracy, so occasional text errors may occur. Report any corrupted tests immediately so we can continue improving.
